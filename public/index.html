Features

Built for teams who ship fast.

✍️
Markdown editor with live preview

Full markdown support — headings, lists, code blocks, links. See exactly how it'll look as you type.

🔗
Shareable public URL

Every project gets a permanent URL at /p/slug. Share with users, embed in docs.

📦
Unlimited projects

One app, many products. Each project has its own entries, slug, and fully independent public page.

🏷
Release type labels

Tag entries as Major, Minor, Patch, or Fix. Each renders with distinct styling on the public page.

📤
Draft before you publish

Write privately, publish when ready. Unpublish any time — gone from the public page instantly.

⚡
Server-rendered public pages

Public changelogs render on the server — blazing fast, SEO-indexed, zero JS required to read.

How it works

First changelog live
in under 10 minutes.

01
Clone & install

Download, run npm install. Zero build step — plain HTML, CSS, and serverless functions.

npm install
02
Connect Turso

Create a free Turso database, paste the URL and token into Vercel env vars. Schema bootstraps itself.

TURSO_URL + TOKEN
03
Push & deploy

Push to GitHub, connect Vercel, done. Every git push triggers a redeploy automatically.

git push → live
