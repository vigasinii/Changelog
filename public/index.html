Features

Everything a changelog needs.

✍️
Markdown editor

Write entries in full markdown — headings, code blocks, lists, links. Live preview in the editor.

🔗
Public shareable URL

Every project gets a permanent URL at /p/your-project. Share it anywhere.

📦
Multi-project

Manage as many projects as you need. Each has its own entries, slug, and public page.

🏷
Release types

Tag each entry as Major, Minor, Patch, or Fix. Styled differently on the public page.

📤
Publish control

Draft your entry privately, publish when ready. Unpublish any time — it disappears from the public page instantly.

⚡
Server-rendered public pages

Public changelogs are rendered on the server — fast, SEO-friendly, no JS needed to view.
